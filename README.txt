==============================================================
  Tekken Revolution Online — Quick Start Guide
==============================================================

If you already have the game installed (with firmware, game
files, updates, license, and EBOOT), then you only need to
apply the patches in these steps:

  Step 2 — Patch RPCS3 (P2PS retry limit)
  Step 6 — Trophy mirror (NPWR04645_01)
  Step 7 — Patched EBOOT.BIN

Everything else (steps 1, 3, 4, 5, 8, 9) can be skipped if
your setup is already complete.

--------------------------------------------------------------
  Full step-by-step (first-time setup)
--------------------------------------------------------------

  1. Install PS3 firmware
  2. Patch RPCS3 binary (1 byte — P2PS retry limit)
  3. Download + install game from Sony CDN
  4. Download + install updates 1.01-1.05
  5. Install .rap license
  6. Create trophy mirror (fixes bogus "343 GB" error)
  7. Install patched EBOOT.BIN (online fixes)
  8. Write recommended per-game config (60 FPS, Vulkan)
  9. RPCN account setup (online login credentials)

--------------------------------------------------------------
  Requirements
--------------------------------------------------------------

  - RPCS3 v0.0.40 or later (https://rpcs3.net)
  - Internet connection (for downloading game + updates)
